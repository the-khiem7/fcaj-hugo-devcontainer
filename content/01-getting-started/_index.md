## Purpose

This page shows the recommended structure for a new section in this template.

## Create a New Section

Create paired English and Vietnamese files in the same folder:

```text
content/my-section/
  _index.md
  _index.vi.md
```

Use stable, lowercase, hyphenated folder names. Keep the front matter shape aligned across languages.

## Next Steps

After creating a section, add child pages as folder bundles when they may need screenshots, diagrams, or attachments later.
